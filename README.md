# PyPlayer
Terminal-based mp3 player with python.

##Installation
With git:
---
`git clone https://github.com/BenceTuri/PyPlayer/tree/main`

Change into the cloned repo:

`cd PyPlayer`

And then just run the script with:

`python3 player.py`
